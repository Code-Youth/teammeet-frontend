Welcome to TeamMeet

What were about

Helping people find the right team mates so everyone of any skill level can have fun playing sports

How it works

Create a game and post it for others to join or search for a game already started

Create a game

Start your own game and tailor it to your liking. You can choose the skill level of the game, when and where it be held, and add any other details!

Find a game

Search for a game that matches your criteria. You filter games by skill level, sport, date, and location!
